# **App Name**: LocalCart

## Core Features:

- Product Catalog: Display products with details like name, description, price, discount, stock, category and locally stored images.
- Shopping Cart: Maintain a backend cart per user, with quantity controls and auto price calculations.
- Checkout Process: Enable COD (Cash on Delivery) checkout with order summary and address management.
- Order Management: Track order status (Pending, Confirmed, Shipped, Delivered) with admin update capabilities.
- User Authentication: Secure user registration and login with JWT and HTTP-only cookies; role-based access control.
- Admin Product Management: Admin interface to add, edit, delete products, upload local product images, and control frontend visibility.
- Search and Filter: Backend-powered search and filtering using MongoDB queries. No AI, no third-party APIs. Example (Normal Logic): Search by product name, Filter by category, Filter by price range, Sort by price / newest

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082), reflecting security and control over local data.
- Background color: Very light gray (#F0F0F0), for a clean, uncluttered user interface.
- Accent color: Muted Violet (#8A2BE2), for interactive elements and key actions.
- Body font: 'PT Sans', a humanist sans-serif font.
- Headline font: 'Space Grotesk', a computerized, techy sans-serif.
- Use simple, modern icons to represent product categories and actions.
- Clean and lightweight layout inspired by Flipkart, emphasizing product visibility and easy navigation.
